Inventory File Validator

Introduction

To assist agencies with preparation of their required DCOI Inventory files, a validation tool to check their inventory csv files is provided below. Agencies' final DCOI inventories must be submitted to OMB via the method listed in the quarterly IDC guidance.

Please refer to the IDC guidance for more information about the fields that must be submitted to OMB.

Validator

This tool helps to find common issues with inventory data CSV files. It may not catch every problem, and OMB may ask you to make further changes or modifications upon submission.

This website does not collect any inventory data. Please refer to the IDC Guidance for instructions on how to submit this data to OMB.
